# How to use this folder

There are three different types of files here:

1) Jupyter notebooks - deriving EOMs and running the TrajOpt code
2) Python scripts - storing user defined libraries
3) pickle files - storing the EOMs

The Jupyter notebooks are the "main" files which shuuld be run. 

There is another directory that stores saved models. This is often essential to get the solution to converge.
